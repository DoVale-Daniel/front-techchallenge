# front-techchallenge

## Install yarn 
https://yarnpkg.com/lang/en/docs/install/#debian-stable

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
